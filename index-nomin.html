VeilMC Website

The VeilMC Network is a Minecraft network with about 200k players registered I work with. I develop Java Plugins for them and I developed a fully operating website which fetches user data from a Redis Database, where data is published with a custom plugin. Saving method is executed asynchronously so it does not affect the game play. It also contains a forum, which is currently using xenforo as a Backend system. It also has a public API for those who aspire to develop any kind of stuff for this network. A live preview is available here. This projectwill not be resold.
